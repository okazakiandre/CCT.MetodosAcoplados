CCT - Métodos acoplados (dependências fixas)
=============================================

- Como esse conhecimento te ajuda?
  - entender porque o acoplamento dificulta os testes
  - aprender a desacoplar
  - vai escrever código melhor

- Mostrar código

- Por que é um problema?
  - o código vira um blocão único
  - métodos grandes que fazem coisas demais
  - testes também ficam grandes e confusos
  - ninguém sabe testar (ou não quer)

- Como resolver
  - use interfaces
  - troque o new por injeção de dependência
  - foque seus testes nas unidades

- Conclusão
  - acoplamento não combina com teste de unidade
  - código acoplado é mais difícil de alterar
  - nunca é tarde para aprender o básico
